Code Editor Client- https://kapilbisht-web.github.io/Code-Editor-Client/

A web-based code editor built with **React.js** and **Monaco Editor**, designed to provide a recruiter-ready IDE-like experience.  
This client connects to the backend via **Socket.IO** for real-time code execution and output display.
